# 📊 Blinkit Sales Dashboard (Excel Project)

## 🔹 Overview
This project is an interactive Excel dashboard built to analyze Blinkit grocery sales data and uncover key business insights.

## 🔹 Key Features
- Total Sales Analysis ($1.2M+)
- Dynamic Filters (Slicers)
- Category-wise & Outlet-wise performance
- Clean and interactive dashboard design

## 🔹 Key Insights
- 🛒 Fruits & Snacks generate highest sales (~$178K)
- 🏪 Tier 1 outlets contribute maximum revenue (~$472K)
- 🥗 Low fat items dominate (~64.6% sales)

## 🔹 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers
